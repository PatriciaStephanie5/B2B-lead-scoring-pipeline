# B2B-lead-scoring-pipeline
Python-based lead scoring system for B2B sales prioritization

## Overview
This project simulates a real-world task performed by a Junior Data Analyst at a B2B company. The goal is to prioritize high-value leads for sales outreach using a data-driven scoring system.

---

## Objective
To build a lead qualification model that ranks companies based on their likelihood toconvert.

---

## Tools used

Python (Pandas)
CSV data processing
Basic scoring algorithms

---

## Process
1. Loaded and cleaned lead dataset
2. Defined scoring criteriabased on:
   - Industry
   - Company size
   - Engagement score
3. Aplied scoring logic using Python
4. Ranked leads by priority
5. Exported results for sales use

---

## STAR Framework

**Situation**
Sales team lacked a structured way to prioritize leads, leading to inefficient outreach.

**Task**
Develop a scoring system to identify high-value prospects.

**Action**
Used Python(Pandas) to clean data, define scoring logic, and rank leads based on business-relevant signals.

**Result**
Generated a prioritized lead list, improving efficiency, and enabling targeted sales outreach.
